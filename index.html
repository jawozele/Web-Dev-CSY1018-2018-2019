Core Purpose

The core purpose of the site will enable users/visitors to find fundamental resources they require in learning about Web Development. The site consists of 5 pages, some of which are integral in aiding beginners to learn the basics of Web Development.

Reviews

There are various virtual learning resources available on the web. Programming beginners may find it quite daunting to narrow down searches for those resources that have been well put together to help interested readers get up to speed.

"The website has selected two key resources from a long list of resources available on the web. The main focus will be on freeCodeCamp and Udemy. These web resources have been selected as they have an abundance of web contents available for beginners."

Glossary

Web development include an extensive lists of jargon and abbreviations that are mostly used in developing responsive sites. The glossary page on this website will enable visitors to understand some important terms used in web development. The page also provides external links to online resources covering web development.

"One of the hardest parts of shifting to a career in the web or product world is the unbelievable amount of industry jargon woven into daily dialogue. To an outsider, the volume of terms and acronyms you need to know to keep up with conversation can be totally mind-numbing. If tech-speak is throwing you for a loop and you're struggling to get familiarized with all the lingo, here's a glossary of some of the more commonly used web design terms to the rescue!"

Jobs

Front-end Web Development focuses on web interfaces, ideally the parts of the website that the user will interact with. When browsing the web, most visible part of the website such as images and text are created using Hyper Text Mark-up Language - HTML.

The jobs page compares two web development jobs and also includes some key information which are relevant to the front-end job seekers.

Tutorials

There are various virtual learning resources available online to begin web development. The course allow users to learn the basics of HTML and CSS. The projects on offer are hands on in a manner that can enable new skills. There are also specialized courses like PHP which are available once learners have a working knowledge of HTML and CSS. The resources also cover the ins and outs of web application development.
